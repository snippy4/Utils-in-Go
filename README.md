# Utils in Go
A collectiobn of utility scripts based off of the Unix terminal commands to use in windows, and some additional scripts for Linux too.

### Prerequisites
None

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/snippy4/Utils-in-Go
   ```
2. add the bin file to PATH
   
3. Use the commands in your cmd or powershell terminal
   ```bash
   hello-go-utils
   ```
